<Meta title="TypeScript/Using record" />

## Replacing `[key: string]`

```ts
// Option 1
interface Score {
  [key: string]: number;
}

// Option 2
type Score = Record<string, number>;

const scoreData: Score = {
  math: 23,
  science: 45,
};
```

## Dictionary as key pair values

```ts
interface Score {
  math: number;
  science: number;
}

// Create empty object
const studentScore: Record<string, Score> = {};

// Create object with values
const studentScore: Record<string, Score> = {
  math: 2,
  science: 3,
};
```

TypeScript does not allow creating empty objects with defined type / interface. Using `Record` can solve this issue.
