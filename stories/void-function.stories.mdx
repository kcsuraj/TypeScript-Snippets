<Meta title="TypeScript/VoidFunction" />

Instead of using `() => void` function type you can use the built-in `VoidFunction`.

```ts
// option 1
const handleCancel: () => void = () => {};

// option 2
const handleCancel: VoidFunction = () => {};
```
